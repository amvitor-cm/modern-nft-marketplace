# NeoNFT - Modern NFT Marketplace Responsive Portfolio Project

## Overview

NeoNFT is a modern, responsive frontend-only NFT marketplace built with vanilla HTML, CSS, and JavaScript. The platform features a clean, professional design with light/dark mode toggle, wallet integration, and comprehensive NFT browsing capabilities.

## Live Demo

[View Live Demo](https://nft-marketplace-yixr.vercel.app)

## Features

- **Modern UI/UX**: Clean, responsive design optimized for all devices
- **Light/Dark Mode**: Toggle between light and dark themes
- **Wallet Integration**: Connect with popular Web3 wallets (MetaMask, WalletConnect, Coinbase)
- **NFT Marketplace**: Browse, view, and interact with digital assets
- **Creator Profiles**: Discover top creators and their collections
- **Advanced Filtering**: Filter NFTs by category, price range, and status
- **NFT Creation**: Upload and create new NFTs with metadata
- **Search Functionality**: Find specific NFTs and collections
- **Mobile Responsive**: Fully optimized for mobile and tablet devices

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with CSS Variables
- **Icons**: Lucide Icons
- **Fonts**: Inter font family
- **Deployment**: Netlify

## Key Sections

1. **Hero Section** - Engaging introduction with call-to-action buttons
2. **Featured Drops** - Highlighted NFT collections with carousel
3. **Trending NFTs** - Grid display of popular digital assets

6. **Explore Page** - Comprehensive NFT browsing with filters
